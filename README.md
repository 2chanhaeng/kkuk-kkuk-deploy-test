## Pages

- `/`: 루트 페이지(전체 키트 조회)
- `/kits`
  - `/new`: 새로운 키트 생성 페이지
  - `/[kitId]`: 특정 키트 페이지
    - `/start`: 특정 키트를 통해 랠리 생성 페이지
- `/my`: 나의 정보
  - layout
  - `/joined`: 참가 중인 랠리 페이지
  - `/completed`: 완료한 랠리 페이지
  - `/created`: 생성한 키트 페이지
  - `/settings`: 개인 정보 수정 페이지
